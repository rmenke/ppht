Detailed Description

A collection of points surrounding a line segment.

The point_set represents a line segment and the image pixels that make up the segment. The segment need not pass through all of the points and may consist of points not in the pixel set.

Establish a "less than" relationship between point sets.

A point set is "less than" another point set if its segment is shorter. An empty point set is smaller than any non-empty point set.
