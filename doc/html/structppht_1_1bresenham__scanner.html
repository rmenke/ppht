Detailed Description

template<std::size_t Ind, int Increment>
struct ppht::bresenham_scanner< Ind, Increment >

A scanner based on Bresenham's line algorithm.

This approach is ideal for this task because it is designed to work on monochrome rasters and minimizes the involvement of the FPU. This class actually implements the four variants of the algorithm by encoding the slope through the template parameters:

octant I: 0 < m < +1, Δx > 0
octant II: +1 < m < +∞, Δy > 0
octant III: -1 > m > -∞, Δy > 0
octant VIII: 0 > m > -1, Δx > 0

All other cases are handled by reversing the direction of the scan.

Template Parameters
Indthe index of the independent field of point_t (0 = X, 1 = Y).
Incrementthe increment or decrement for the dependent field.
